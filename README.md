# employeemanager
This application supports basic CRUD functionality developed using Spring Boot as backend and Angular 10 as front-end.

# Features <br/>
Add an employee <br/>
Update employee <br/>
View all employees <br/>
Delete an employee<br/>

# More details
Database used - MySQL <br/>
API - REST API
